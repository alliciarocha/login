### ‚ú® Key Features

* **Appealing Design:** Clean and minimalist interface featuring vibrant colors and a blur effect.
* **Dynamic Video Background:** Uses a looped video to create an immersive visual experience.
* **Toggleable Forms:** Smooth transition between the **Sign In** and **Create An Account** views.
* **Animated Input Fields:** Placeholder effect that moves up upon focus, enhancing usability.
* **Responsiveness:** The layout adapts perfectly to different screen sizes (desktop, tablet, and mobile).
* **Social Login:** Quick login icons for Google, Facebook, and Apple.
